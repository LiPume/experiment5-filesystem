
# 操作系统实验五：简单文件系统的实现。

本项目用于实现一个基于虚拟磁盘的简单文件系统，支持目录管理、文件创建删除、文件打开关闭、文件读写、退出保存等基本功能。

## 项目目标

在内存中开辟一块虚拟磁盘空间，模拟一个单用户文件系统，并在其上实现以下功能：

- 文件系统初始化与格式化
- FAT 表管理磁盘空间
- 多级目录管理
- 文件创建与删除
- 文件打开与关闭
- 文件读写
- 文件系统退出与保存

cd experiment5-filesystem
2. 本地编译
